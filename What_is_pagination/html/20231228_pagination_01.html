Pagination

Pagination이란?

콘텐츠가 여러개 쌓이다 보면 스크롤이 길어질 수 있는데 그걸 방지하기 위해서 콘텐츠를 적당한 묶음으로 나누고
숫자를 매겨 페이지 단위로 이동이 가능하게끔 하는 걸 말한다.

메인 슬라이드와 Pagination

Pagination이 무슨 뜻인지 정확하게 잘 몰랐는데 메인 슬라이드가 작동하는 방식이 이것과 같다. 대신 메인 슬라이드는
정해진 시간마다 슬라이드가 자동으로 넘어가는 기능 정도가 추가적으로 붙어있는듯.

메인 슬라이드를 만들기 위해 지금 알고있는 부분들

우선 슬라이드를 보여주고자 하는 영역에 하나씩 나타나도록 배치한다. 슬라이드의 순서와 위치를 나타내는 인디케이터도
슬라이드의 적절한 위치에 배치한다.

이제 스크립트를 작성해야 하는데, 슬라이드의 개수와 인디케이터의 개수 그리고 index가 일치하는 점을 이용해서, 예를 들어
0번째 인디케이터를 클릭했다면 0번째 슬라이드가 슬라이드의 영역에 나타나도록 해야 하는 것이다. 여기까지는 좋은데 어떻게
슬라이드를 해당 영역에 위치시키느냐가 문제가 되는 부분이였다. 제일 나아보이는 방법이 해당 슬라이드가 완벽하게 보여지는
지점까지 슬라이드 전체를 이동시키는 것인데 모니터나 기기의 너비도 각각이라 이 부분을 어떻게 계산하느냐에서 막혀있다.
